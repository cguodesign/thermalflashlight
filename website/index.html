Applications

Thermal imaging can be used to document heat/AC leaks from insulation gaps on a building's facade, reveal warmer ground water inflows (either fresh or chemical-laden) or "thermal pollution" from industrial processes entering ocean-temperature waterbodies, as well as identify areas on the human body experiencing infection or stress (includes epidemiological applications).

Our Contribution

The previous version of thermal flashlight could only be used in a certain range of temperature. The color would looks like the same one if the temperature is above/below the range. Beside that, to calibrate the temperatural range, user has to connect to the computer and modify the program.

Within our design, we use LED module to help user in the calibration phase. The LED module has multiple buttons that could be used in setting temperatural range. The module is directly hooked up to Arduino board, which would be easy for the first time project user.
